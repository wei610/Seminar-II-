# 具冒充者與欺騙攻擊偵測之語言辨識
# 演講者 許正欣教授
# 演講日期 2026/04/28
# 筆記內容
# 一、語者辨識
## 語者辨識系統
●冒充者(Imposter<BR>
●目標語者(Target)<BR>
●欺騙攻擊(Spoofing attacls)<BR>
## 欺騙攻擊
以人為假造聲音，對語者辨識系統進行攻擊。
## 物理性訪問攻擊
透過曼克風進行攻擊。
## 邏輯性訪問攻擊
繞過麥克風，直接攻擊系統。
## 語者辨識類型
●語者驗證<BR>
●語者辨識
## 語者辨識
●本文相關：限制辨識之語劇內容，通常效果較好。<br>
●本文無關：不管語句的內容。(br)
# 二、語音特徵擷取
●常數Q倒頻譜係數：結合常數Q轉換和倒頻譜過程所得到的語音特徵，對聲音上變化更敏銳。
●i-vector：使用MFCC進行擷取，需要使用非註冊語者語料來建立通用背景模型統計語者特徵分布的模型。
# 三、分數正規劃
對模型的輸出分數作正規化，使目標語者跟冒充者之間的分數分布能夠拉開，期望更容易找出適用門檻值。
# 四、混淆矩陣
欺騙攻擊偵測系統的評估：<BR>
●正確率<BR>
●真陽率<BR>
●偽陽率<BR>
語者辨識系統的評估：<BR>
●正確率=辨識率<BR>
●真陽率=目標語者通過率<BR>
●偽陽率=1-冒充者阻擋率<BR>
